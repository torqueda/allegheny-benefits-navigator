677.5 Withdrawn Applications

An applicant may voluntarily withdraw their application at any time before the eligibility decision is made. An applicant can verbally voluntarily withdraw their application. If the applicant withdraws the application and gives a reason, the CAO must narrate the reason.

A withdrawn application can be rejected through the Application Processing module or the Case Processing module. If the application has been pushed to Case Processing, the worker must run eligibility and a supervisor must override the eligibility decision using the rejection code for a withdrawn application.

Appendix C: PROMISe Inquiry System

Overview

The PROMISe inquiry system offers CAOs a list of names, addresses, and telephone numbers of MA providers. The list can be sorted by county or by zip code. Listed providers have been in the MA program in the past 18 months.

Sign On/Sign Off

For information on providers, click on the PROMISe icon, which is on each CAO computer desktop. Log onto the PROMISe Web page, and then click on âProvider Searchâ on the left side of the mail page. You can select up to five fields or just the county and type of provider.

Inquiry

The PROMISe inquiry allows inquiry by county code and ZIP code of providers by provider type and specialty code. An entry is always required for the county code and the print site.
